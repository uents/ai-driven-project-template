# 契約定義

## 概要

本ディレクトリは、サービス間の契約（Contract）を定義するファイルを管理します。

## 目的

- サービス間のインターフェースを明確にする
- 契約に基づいたコード生成・テスト生成を可能にする
- 契約の変更を追跡可能にする

## 構成

| ディレクトリ | 内容 |
|---|---|
| `api/` | OpenAPI仕様によるREST API定義 |
| `events/` | EventBridgeイベントのJSON Schema定義 |
| `database/` | DynamoDBテーブル定義・アクセスパターン |

## 利用ルール

- 契約の変更は必ずPRでレビューを受けること
- 後方互換性を破壊する変更は事前にチームで合意すること
- 契約ファイルを更新したら、関連するコード・テストも同時に更新すること
