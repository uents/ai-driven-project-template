# E2Eテスト

## 概要

実際のAPIエンドポイントに対してエンドツーエンドのシナリオを検証するテストです。

## テスト戦略

- デプロイ済みのステージング環境に対して実行する
- 実際のHTTPリクエストを送信し、レスポンスを検証する
- テスト実行後にテストデータをクリーンアップする

## 実行方法

```bash
# E2Eテストの実行（ステージング環境）
API_BASE_URL=https://staging-api.example.com task test:e2e

# 特定のテストファイルのみ実行
pytest tests/e2e/test_api_e2e.py -v
```

## 前提条件

- ステージング環境がデプロイ済みであること
- `API_BASE_URL` 環境変数が設定されていること
- APIキーが設定されていること
