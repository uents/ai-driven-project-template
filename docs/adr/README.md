# ADR（Architecture Decision Records）

## ADRとは

アーキテクチャ上の重要な意思決定を記録するドキュメントです。
なぜその技術・設計を選択したかの理由とトレードオフを明示します。

## テンプレート

```markdown
# ADR-XXXX: {タイトル}

## ステータス

承認済 / 提案中 / 廃止

## コンテキスト

{意思決定が必要になった背景・状況}

## 決定

{採用した技術・設計方針}

## 理由

{この決定に至った理由}

## トレードオフ

{この決定によるメリット・デメリット}

## 代替案

{検討した他の選択肢とその評価}
```
