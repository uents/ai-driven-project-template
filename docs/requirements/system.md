# システム要件

## 概要

本ドキュメントは、システムの非機能要件およびシステム要件を定義する。

## 性能要件

| 項目 | 要件 |
|---|---|
| API応答時間 | 95パーセンタイルで500ms以内 |
| スループット | 1000リクエスト/秒 |
| Lambda同時実行数 | 最大500 |
| DynamoDB読み取りキャパシティ | オンデマンドモード |

## 可用性要件

| 項目 | 要件 |
|---|---|
| 稼働率 | 99.9%（月間ダウンタイム約43分以内） |
| リージョン | ap-northeast-1（東京） |
| DR方式 | DynamoDBのポイントインタイムリカバリ |

## セキュリティ要件

- 通信はすべてHTTPSで暗号化すること
- APIアクセスはAPI Keyまたは認証トークンで制御すること
- 機密情報はAWS Secrets Managerで管理すること
- DynamoDBの保存時暗号化を有効化すること

## 運用要件

- ログはCloudWatch Logsに集約すること
- メトリクスはCloudWatch Metricsで監視すること
- アラートはCloudWatch Alarmsで通知すること
- デプロイはGitHub Actionsによる自動デプロイとすること
