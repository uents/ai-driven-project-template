# 用語集

本プロジェクトで使用するドメイン用語の定義です。コード内の命名はこの用語集に基づいて統一してください。

| 日本語 | 英語（コード内） | 説明 |
|---|---|---|
| ユーザー | User | システムを利用するエンドユーザー |
| ユーザーID | user_id | ユーザーを一意に識別するID（UUID v4） |
| ユーザー名 | name | ユーザーの表示名 |
| メールアドレス | email | ユーザーのメールアドレス（一意制約あり） |
| プロフィール | profile | ユーザーの詳細情報 |
| ユーザー作成イベント | UserCreated | ユーザー作成時に発行されるイベント |
| アクセスパターン | access_pattern | DynamoDBのデータアクセス方法の定義 |
| パーティションキー | PK | DynamoDBのパーティションキー |
| ソートキー | SK | DynamoDBのソートキー |
